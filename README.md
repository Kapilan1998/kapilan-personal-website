### Core Framework
- **React 18** with **TypeScript**

- **Tailwind CSS** for utility-first styling
- **shadcn/ui** component library (built on Radix UI)
- **Framer Motion** for smooth animations
- **React Three Fiber** & **Three.js** for 3D elements

### State & Data Management
- **TanStack React Query** for server state and API caching
- **React Hook Form** with **Zod** validation
- **Context API** for theme/global state

### UI Components & Libraries
- **Radix UI Primitives** (Dialog, Dropdown, Tabs, Toast, etc.)
- **Recharts** for data visualization
- **Lucide React** for icons
- **Embla Carousel** for sliders
- **Sonner** for toast notifications

### Development Tools
- **ESLint** + **TypeScript ESLint** for code quality
- **PostCSS** + **Autoprefixer** for CSS processing
- **Git** for version control
